Course Curriculum

  • Revision of Python
    We will quickly revise python concepts. This course requires that you are already comfortable with python. Please enroll in our python course if you don't have prior experience with python.
  • Mathematics & Machine Learning
    Machine Learning relies on mathematics concepts such as probability, vectors, statistics, calculus and matrices. While you may have learned these before, we will do a quick revision of these important maths concepts.
  • Data Pre-Processing
    First step of machine learning is analysis of data about a given situation, the data could be details of passengers on an airplane or details of movies that a person likes or even a chat conversation on a whatsapp group. We will be analysing a data set using the popular python libraries pandas and matplot lib. Post analysis we will prepare our data using techniques such as feature selection, scaling, encoding and filling missing values.
  • House Price Analysis
    Hello world program for machine learning, we will use a technique known as linear regression to predict price of houses based on various features of the house such as number of rooms, amenities etc.
  • Supervised Learning
    One of most widely used and powerful category of machine learning algorithms are referred to as supervised learning. Supervised learning relies determining the outcome of a given situation, like the toss of a coin from a given setof pre-existing outcomes. You will study logistic regression, K-NN and Gausian Naive Bayes algorithms and implement them using the scikit library.
  • Unsupervised Learning
    You will learn how to find patterns and structures in situations where there is no definite set of fixed outputs. An example is determining the colors in an image, since colors are infinite in number we can use an unsupervised learning algorithm like K-Means to determine prominent colors in a given image.
  • Final Project
    You will be assigned a project to work on that you can submit in your university or show to companies during interviews.
