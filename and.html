Course Curriculum

  • Commencement and Installation
    We will be kicking off your course with a session by TechLedge Head Coaches and set up the environment for building Android Apps.
  • Introduction To Android
    Learn basics of Android Development in JAVA
  • Android Framework & Architecture
    Introduction to ART / Dex Files & The Concept Of Activity
  • Activity
    Deep dive into App Activity
  • Android UI Concepts : Layouts & Views
    Know about pre-build UI components in Android.
  • Advanced UI View & Registeration Form
    Understand the best practices about User Interaction
  • Menu's
    Learn about more action options for a view
  • List View & AutoComplete Text View
    Discuss about Intent and Listings for Apps
  • Fragment
    Understand about behavior or a portion of user interface in an Activity
  • Prefrences & Dialogs
    Explore Shared Preferences and Data options
  • Data Storage - Internal & Extrenal
    Learn about Data needs and storage
  • Introduction To SQLite
    Learn about SQL and databases in Android
  • Database in Android
    Integration of Android App with Database
  • Threads
    Understand Threading in Android
  • Material Design
    Explore Material Design to create fascinating interfaces.
  • Service
    Know about data exchange in the form of restful API i.e. JSON/XML using HTTP calls (POST, GET, PUT, and DELETE).
