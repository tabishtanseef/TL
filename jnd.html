Course Curriculum

  • GIT
    Introduction to the world's most popular Version Control System Git, how to create a repository, add and commit files. We will learn creating profile on github, configuring a remote repository and pushing code to that remote repository. Companies love it when you know git.
  • Basics in OOPS
    Master the art of dividing problems into interaction of classes and objects and how do they appear in real life. Understand usages of keywords like static, new, abstract, super etc., constructors, interfaces, wrapper classes and 4 pillars of OOPS.
  • Advanced Concepts in OOPS
    Learn advanced concepts of Exception Handling, Generics, File handling, Reading and Writing to files, Data structures using Collections Framework and Multi threading to get a strong hands-on coding experience in JAVA.
  • Guest Lecture on KOTLIN
    Dive into the new Android Programming language by Google - Kotlin. Learn to make Hello World! program in Kotlin programming language and get insights on how is it changing the world.
  • Introduction to Android
    Learn basics of Android Development in JAVA, Android Framework and Architecture, and setup Android Studio
  • Activities
    Introduction to ART / Dex Files \u0026 The Concept Of Activity and Deep dive into App Activity
  • Android UI Concepts
    Layouts and Views, Advanced UI View and Registration Form, Menus, List View, Recycler View and Autocomplete Text View, Fragments, Fascinating interfaces using Material Design.
  • Databases in Android
    Fragments, Preferences and Dialogs, Internal and External needs, Introduction to SQLite and interaction with Backend data through logic in JAVA.
  • REST Service and Broadcast Receiver
    Know about data exchange in the form of restful API i.e. JSON/XML using HTTP calls (POST, GET, PUT, and DELETE). Learn broadcasting and receiving messages, Notifications to use, Content Provider.
  • Google Maps and Cloud
    Google Map integration, Introduction to ML using GoogleCloud, Image Analysis using Google AI.
